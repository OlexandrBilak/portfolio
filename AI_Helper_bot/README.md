# AI Помічник

**AI Помічник** — це інтелектуальний Telegram-бот, який допомагає користувачам обробляти та трансформувати інформацію простими словами. Бот поєднує можливості розпізнавання голосу, аналізу відео та тексту, перекладу та пояснення складних тем.

---

## Основні можливості

### 🧠 Перетворення
- Перетворює голосові повідомлення у текст.
- Аналізує аудіо та відео, витягуючи текстову інформацію.
- Ідеально підходить для розшифровки лекцій, подкастів, відеоуроків та інтерв’ю.
- Використовує технологію **Whisper** для точного розпізнавання мови та **GPT** для обробки та структурування тексту.

### 🌍 Переклад
- Автоматично визначає мову введеного тексту.
- Перекладає тексти між будь-якими мовами.
- Якщо текст українською — запитує, на яку мову перекласти.
- Зберігає структуру тексту: абзаци, списки, форматування.
- Підходить для навчання, роботи та комунікації з іноземними мовами.

### 📚 Навчання
- Пояснює складні теми простими словами.
- Дає приклади та детальні пояснення термінів.
- Підходить для студентів, спеціалістів та всіх, хто хоче швидко зрозуміти складну інформацію.
- Підтримує контекст попередніх повідомлень, щоб навчання було більш структуроване.

---

## Як працює

Бот обробляє повідомлення у трьох режимах:

1. **Перетворення**  
   Користувач надсилає голосове повідомлення або відео → бот розпізнає аудіо → перетворює у структурований текст.

2. **Переклад**  
   Користувач надсилає текст → бот визначає мову → перекладає на потрібну мову, зберігаючи форматування.

3. **Навчання**  
   Користувач задає складне питання або тему → бот пояснює її максимально просто і зрозуміло, наводячи приклади.

---

## Особливості
- Зберігає коротку історію останніх повідомлень для кращого контексту.
- Використовує сучасні AI-технології OpenAI: Whisper та GPT-4o-mini.
- Простий і зрозумілий інтерфейс з кнопками для вибору режиму роботи.
- Працює з аудіо, відео та текстом одночасно.
- Не відповідає на зайві питання в режимі навчання — фокус на поясненні тем.

---

## Контекст і обмеження
- Бот пам’ятає тільки останні 5 повідомлень у поточному сеансі.
- В режимі **Переклад** бот не змінює зміст тексту.
- В режимі **Навчання** бот пояснює лише факти, не вигадує нові дані.
